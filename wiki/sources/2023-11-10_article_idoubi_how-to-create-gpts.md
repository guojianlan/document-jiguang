# 如何创建 GPTs 应用

## 一句话摘要

GPTs 发布次日的入门教程，演示用 GPT Builder 无代码创建 GPTs + 通过 Actions 调 OpenAPI 3.1.0 第三方接口的全流程。

## 关键事实

- GPTs 在 OpenAI 第一届开发者大会（2023-11-06）发布
- 内置能力：Web Browsing、DALL-E 3、Code Interpreter、私有文档、第三方 API
- 通过 OpenAPI 3.1.0 Schema 配置 Actions 调用外部摘要 API
- 以"知了阅读"为例完整演示创建流程

## 作者观点

- GPTs Store 是 AI 时代的 App Store，普通开发者有机会创建超级应用
- 工业时代 → AI 时代的转换窗口窄，要尽早学

## 我的判断

- 这是 idoubi 系列的起点——他后续大量产品（GPTs Works、ThinkAny 等）都是这个 GPTs 节点开始的，**作为 timeline 起点价值高**
- GPTs Actions 是后来 [[MCP]] 的早期形态——OpenAPI Schema 描述工具，AI 自动调用，思路一脉相承
