# 2025，MCP、Agent、AI Coding

## 一句话摘要

idoubi 2025 年终总览——MCP.so 11M 年访问 + 1.2 万美金广告、《这就是 MCP》出版、Agent 三大件论、AI Coding 两方向，是当年信息密度最高的一篇。

## 关键事实

- **MCP.so**：年访问 11M + 新用户 1.5M，4 月单月 2.7M，品牌广告 1.2 万美金，被 a16z 引用
- **MCPRouter**：K8s 云端 MCP 托管 + 多服务整合，解决 MCP 上手门槛
- **Agent 三大件**：Framework（LangGraph / AutoGen 过底层）+ Tools（MCPRouter）+ Sandbox（e2b / Webcontainers 成本高）
- **AI Coding 两方向**：垂类深耕（CopyWeb 网页复刻）/ 模板增量（ShipAny 做 Coding Agent 底座）
- 《这就是 MCP》3 个月销 3700+ 册

## 作者观点

- Agent Skills 与 MCP 互补不替代——Skills 给 Agent 看说明书，MCP 给工具写说明书
- Manus 成功在工程细节（sandbox + todo.md），复刻容易细节难
- 2026 Agent 瓶颈在 Sandbox 接入

## 我的判断

- "Agent 三大件"框架（Framework / Tools / Sandbox）是个**清晰好用的认知模型**，适合直接进 [[AI Agent]] concept 页
- MCP.so 的 11M 流量 + 1.2 万美金广告——**这是 idoubi 全部产品里 ROI 最好的一个**，对应了"MCP 是平台机会"判断的真实兑现
- "Skills 给 Agent 看 / MCP 给工具写"这个区分非常精准，可作为 [[MCP]] entity 与 Skills 关系段的标准说法
- 与 [[2025-01-01_article_idoubi_my-ai-2024]] 的"MRR 10k 目标" 对照：2025 年终没明示是否达成 10k，**但 MCP.so 单广告就 1.2 万美金，暗示已经超额**
