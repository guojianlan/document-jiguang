# 我开源了一个 Sora AI 视频生成器模板，却被人拿去割韭菜

## 一句话摘要

注册 sora.fm 蹭 Sora 热度一周开源 + Mock API 期货——结果项目被他人挂"有 OpenAI 账号"的虚假宣传割韭菜，引发对独立开发者道德底线的反思。

## 关键事实

- sora.fm 注册 + 一周开源，2 天 GitHub 300+ Stars，搜索排名第一
- 用 Mock 接口诚实标注"等待 OpenAI 正式 API"
- 被他人盗用做欺诈宣传，澄清评论被删

## 作者观点

- 套壳无创新可接受，欺诈无底线
- 借《三体》："我点燃了火，却无法控制它"

## 我的判断

- 与 [[2024-01-30_article_idoubi_ai-cover-generator]] 是同一套路（蹭热点 + 模板复用），但揭示了套路的代价：**开源做大后失控**
- "诚信底线"这个观点在 2024 年中文 AI 圈很稀缺，可作为 艾逗笔 entity 的关键人设标签
