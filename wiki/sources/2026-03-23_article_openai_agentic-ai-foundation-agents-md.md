## 一句话摘要

这篇文章最值得关注的点不是基金会本身，而是 AGENTS.md 正在成为跨工具、跨平台的 agent 协作标准。

## 核心观点

- Agent 进入生产后，需要共享、可移植的项目级指令标准
- AGENTS.md 已经从内部做法走向开放标准
- agent 生态正在朝互操作和中立标准收敛

## 关键事实 / 数据 / 论据

- OpenAI 表示 AGENTS.md 自 2025 年 8 月发布以来已被超过 60,000 个开源项目和 agent framework 采用
- 文章提到 Codex、Cursor、GitHub Copilot、VS Code 等多个工具生态
- AAIF 由 OpenAI、Anthropic、Google、Microsoft、AWS 等参与

## 方法 / 框架 / 流程

- 用统一 Markdown 格式为 agent 提供项目上下文
- 把行为约束写进仓库，而不是只写在 prompt 里
