# 我用 AI 辅助编程，效率提升 x 倍

## 一句话摘要

Cursor 主力 + 8 大场景的 AI 辅助编程实战——重构 2 小时变 15 分钟、多语言翻译效率 10 倍，同时明确反对"AI 淘汰程序员"叙事。

## 关键事实

- 核心快捷键：Tab 智能补全 / Ctrl+L 对话检索 / Ctrl+K 原地编辑 / Ctrl+I 多文件 Composer
- 高频场景：自动补全 / Debug / 写前端（截图生成）/ 重构 / 多语言翻译
- Cursor 优势：基于 VS Code 生态 + 自训练补全 + 一站式规划
- 工具生态：Windsurf / Continue / Cline / screenshot-to-code，**GitHub Copilot 已过时**
- 零基础替代：Composer / Bolt.new / Claude / v0.dev

## 作者观点

- AI 不会淘汰程序员——想象力 / 创造力 / 架构 / 审美 AI 替代不了
- Demo ≠ 产品，社交网红项目多为玩具级
- 把 AI 工具比作"倚天剑屠龙刀"——最终练成"乾坤大挪移"的还是你
