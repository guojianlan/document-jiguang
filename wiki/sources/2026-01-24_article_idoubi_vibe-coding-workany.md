# 我 Vibe Coding 一周，做了个桌面 Agent

## 一句话摘要

Claude Code 全权驾驭 + idoubi 零代码指挥——一周做出 WorkAny 桌面 Agent MVP（Tauri + Hono + Claude Agent SDK），开源 1.4k Stars。

## 关键事实

- 一周流程：初念 → Tauri+Hono 架构 → Claude Agent SDK → UI → 自定义模型/MCP/Skills → Sandbox → 多平台打包 → 开源
- 成本曲线：OpenRouter $110 → 美国 IP → Claude Pro → Claude Max（遇限频）
- idoubi 角色：需求 + 测试，不写代码
- 用户反馈迭代：node 环境引导 / Mac 签名 / 精简版 20MB / GitHub Action 跨平台构建
- 1.4k GitHub Stars

## 作者观点

- 技术平权时代，**产品 sense + 审美 > 代码能力**
- idoubi 角色变成"建筑师"而非"泥瓦匠"
- 优秀程序员不被 AI 淘汰，AI 是协作工具

## 我的判断

- 这是 idoubi 第一次真正的"全自动 Vibe Coding"——与 1 年前 [[2024-12-16_article_idoubi_use-ai-coding-copilot]] 的"Cursor 辅助"对照，呈现 **辅助 → 主导** 的范式跃迁
- "成本曲线" $110 + Claude Max 顶配限频——是个真实的"Vibe Coding 不便宜"信号，与那些"AI 让人人零成本写代码"的叙事矛盾
- "建筑师 vs 泥瓦匠"的比喻很准——可以补到 [[AI Vibecoding]] concept 的 Phase 4-5 阶段说明
