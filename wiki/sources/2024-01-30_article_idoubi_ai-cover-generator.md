# 我写了一个 AI 红包封面生成器

## 一句话摘要

基于 aiwallpaper 项目改造一小时上线 AI 红包封面生成器，开源后单日 UV 超 GPTs Works，证明"踩节日窗口 + 改 prompt + 开源 SEO"的组合拳。

## 关键事实

- 改造 aiwallpaper：仅改 prompt + 样式，1 小时 MVP
- 技术栈：NextJS / TailwindCSS / Supabase / DALL-E 3 / Clerk / S3 / Stripe
- 首周 GitHub 400+ Stars，单日 UV 超过 gpts.works

## 作者观点

- 开源是低成本曝光的最有效方式
- 时机驱动 > 完美打磨——春节窗口比产品深度更重要

## 我的判断

- 这是 idoubi 标准套路的最纯粹版本——同样的代码模板复用 N 次（aiwallpaper → cover → sora.fm 等），只换垂直主题
- 反复印证：**应用层模板复用 + 节日营销** 是 2024 年独立开发者的稳定打法
