# 我的新书《这就是 MCP》出版了

## 一句话摘要

第一本系统讲 MCP 的中文图书出版——签名版预售 500 本当日京东计算机日榜第一，3 个月销售 3700 余册，但版税仅 2.3 万、ROI 未回本。

## 关键事实

- 6 章：原理 / 架构 / 服务器开发 / 客户端开发 / 应用场景 / 生态
- 融合 idoubi 1.5 年的项目案例（flomo-mcp / heybeauty-mcp / thinkany / podlm）
- 13 位业界领袖推荐（硅基流动、Manus、Same、ChatWise、Cherry Studio 等）
- 3 个月销 3700 余册，自购 1500 本回赠社区，**版税 2.3 万 ROI 未回本**

## 作者观点

- MCP 是 AI 互联网三年最大平台型机会
- 出书是对积累的系统化总结 + 对他人的意义贡献

## 我的判断

- "ROI 未回本"这个数据点很重要——揭示中文技术书籍市场的真实天花板，与 [[2024-05-22_article_idoubi_ai-search-engine]] 的"支付率 0.03%" 形成对照
- 但出书的真正回报不是版税，是**强化了 idoubi 作为"MCP 头部布道者"的身份资产**——这是后来 mcp.so 持续高流量的根因
- "自购 1500 本回赠社区" 是聪明的影响力运营动作，可作为 艾逗笔 entity"影响力变现"行为模式的样本
