# 出海第一周，我的 GPTs 导航站火了

## 一句话摘要

GPTs Works 导航站从周末 MVP 到一周 11.8k UV、Google "chatgpt gpts" 第一名的完整复盘——示范"3 天上线 + 开源 + 多平台传播"的独立开发冷启动套路。

## 关键事实

- 周六 NextJS 搭 Web、周日 FastAPI + Zilliz 加语义检索
- GPTs Hunter 数据 5000+ 条，周二开源，一周 658 Stars / 80 Forks
- 周二晚获 monica.im 创始人赞助，三天变现
- 多平台扩散：Twitter / 微博 / 微信 / GitHub / V2EX

## 作者观点

- 出海初期 SEO + 时机 > 完美产品打磨
- 快速迭代 + 社群验证胜过长期闭门

## 我的判断

- 这是独立开发者"完美开局"的样板——但要注意 idoubi 自己 2024 年终复盘里说 GPTs Works 后期没法长期变现（仅赚到一次性赞助），**短期爆款 ≠ 可持续**
- 与 [[2023-11-24_article_idoubi_I-quit-from-tencent]] 是同一段时期的成功案例链——对裸辞决定有正向反馈作用
