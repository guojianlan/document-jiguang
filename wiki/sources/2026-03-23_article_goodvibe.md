# GoodVibe: Security-by-Vibe for LLM-Based Code Generation

## 来源信息

- 原文链接：https://arxiv.org/abs/2602.10778
- 作者：arXiv 论文页
- 发布平台：arXiv

## 一句话摘要

这篇论文关注的是如何在不牺牲太多通用能力的前提下，提升 LLM 生成代码的安全性。

## 核心观点

- 安全性可以被显式优化，而不是默认寄希望于模型“自然写对”
- 代码生成中的安全问题需要独立对待
- Vibecoding 进入生产后，安全质量会成为关键分水岭

## 关键事实 / 数据 / 论据

- 论文摘要提到安全性相对基础模型可明显改善
- 目标是提升安全而不显著伤害通用能力
- 属于“AI 辅助编码安全化”的研究方向

## 方法 / 框架 / 流程

- 把安全约束显式注入到代码生成流程
- 以安全为目标优化模型行为
