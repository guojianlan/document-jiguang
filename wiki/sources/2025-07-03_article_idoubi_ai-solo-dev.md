# AI 时代如何做独立开发

## 一句话摘要

idoubi 独立开发方法论的集大成——6 大感悟 + 5 步 SOP + 3 大增长手段 + 4 大创业方向，是把过去 2 年实践规律化的"操作手册"。

## 关键事实

- **6 大感悟**：天下武功唯快不破 / 快不如精长期主义 / 梦想大切入小 / 自己造血 / 流量为王 / 开心最重要
- **5 步 SOP**：Ideation → Development → Launch → Growth → Monetization；技术栈 NextJS / Supabase / Stripe / Cloudflare
- **3 大增长**：ProductHunt 打榜（最快冷启动）/ 程序化 SEO（长线） / AI Wrapper（套壳追热点）
- **4 大方向**：AI Coding（最商业化）/ Agent / Agent Infra（卖铲子） / MCP（最大平台机会）

## 作者观点

- 90% AI 应用层无技术壁垒，关键在 SEO + 营销
- "先垂再通、农村包围城市"
- AI Coding + Agent 是 2025 最有前景赛道

## 我的判断

- 这是 艾逗笔 entity 最重要的方法论文章——所有零散观察在这里被收敛成"独立开发 SOP"
- "天下武功唯快不破" + "快不如精" 是对立统一——idoubi 自己 2024 年就是被 ThinkAny 教训到要"快不如精"
- "卖铲子（Agent Infra）"方向是个真洞察——印证了后来他自己做 [[FastClaw]] / [[Open Agent SDK]] 的转向
- "AI Wrapper 套壳追热点" + "影响力变现"  这两个方法论合起来，揭示了一个隐含逻辑：**独立开发不是做产品，是做"短平快现金流 + 长期影响力资产"两件事**
