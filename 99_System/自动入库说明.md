# 自动入库说明

现在这个仓库支持基础的自动分类入库。

## 能做什么

传入一个链接或本地文件路径后，脚本会自动：

- 判断它是 `article`、`link`、`video`、`pdf` 还是 `document`
- 把本地文件按需复制到对应目录
- 在 `02_Notes/SourceNotes` 下生成一份来源笔记
- 生成统一命名的 Markdown 文件，便于后续继续加工

## 当前分类规则

### URL

- `YouTube / Bilibili / Vimeo / Loom` 识别为 `video`
- 以 `.pdf` 结尾的链接识别为 `pdf`
- `Substack / Medium / 微信文章 / Zhihu 专栏 / arXiv` 识别为 `article`
- 其他网址默认识别为 `link`

### 本地文件

- `.pdf` 识别为 `pdf`
- `.mp4 .mov .m4v .avi .mkv .webm` 识别为 `video`
- `.doc .docx .xls .xlsx .ppt .pptx .csv .txt .md .rtf` 识别为 `document`
- 其他未知类型先进入 `unknown`

## 用法

在仓库根目录执行：

```bash
python3 scripts/intake_source.py "https://example.com/article"
```

如果是本地文件，并且希望复制进知识库：

```bash
python3 scripts/intake_source.py "/path/to/file.pdf" --copy-file
```

如果你已经知道主题，也可以一起写入：

```bash
python3 scripts/intake_source.py "https://www.youtube.com/watch?v=demo" --topic "AI Agent"
```

如果标题需要手动指定：

```bash
python3 scripts/intake_source.py "/path/to/demo.mp4" --title "Anthropic Agent Demo" --copy-file
```

## 输出位置

- 来源文件：
  - `01_Sources/Articles`
  - `01_Sources/Links`
  - `01_Sources/Videos`
  - `01_Sources/PDFs`
  - `01_Sources/Documents`
- 来源笔记：
  - `02_Notes/SourceNotes`

## 当前边界

这个脚本现在解决的是“自动分发和建档”，还没有直接做全文摘要。

也就是说它会先把东西放对位置，再生成基础笔记。后续你把这条笔记或原始内容发给我，我继续帮你做：

- 摘要
- 关键信息提取
- 事实和观点区分
- 并入主题笔记
- 生成分享文档
