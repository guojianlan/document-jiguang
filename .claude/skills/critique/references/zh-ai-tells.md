## 一、命中即扣分的硬规则（每命中 1 项扣 0.5 分，上限扣 2）

### 1. "而是"句式（含变体）

> 不是…而是 / 并非…而是 / 不在…而在 / 不只是…更是

- 全文 ≤ 1 次。原文已含且自然可以保留，但**新增的"而是"句一律改写**为直接陈述。
- 改写策略：去掉前半句否定，直接给后半句的判断。

### 2. 协作 / 教学路标词

> 接下来我们将 / 我们先来看 / 下面我们 / 让我们一起 / 不妨一起 / 接下来我会

- 命中即删。中文文章不需要主持式串场。
- 替代：直接进入下一段内容。

### 3. 客服 / AI 身份口吻

> 作为 AI / 截至我的知识 / 希望这能帮助你 / 当然 / 没问题 / 好的

- 命中即删，零容忍。

### 4. 冒号（`：`）超额

- 1500-3000 字正文 **0-2 个**为目标，硬上限 ≤ 2。
- 仅允许直接引语提示语（"他说："）。
- 其他场景命中即改写为完整句子。

### 5. 二人称连续使用

- 默认不用"你/你会"。
- 确需读者代入只允许 1 次"试想"或"当你"句式。
- 禁止连续两句出现第二人称。

### 6. "一段一标题"

- 使用 `##` / `###` 时，每个标题下至少覆盖 2 段正文。
- 仅 1 段则下沉为该段段首句。

## 二、AI 高频隐喻词（比喻义命中即改写，字面义不限）

> 噪音 / 背景噪音 / 信号 / 底色 / 光谱 / 滤镜 / 解药 / 土壤 / 基因 / 拼图 / 镜像 / 路径 / 尺子 / 标尺 / 横截面 / 切面 / 切片 / 透镜 / 棱镜 / 窗口 / 缩影

替代策略：把比喻还原成具体动作、对象、机制。
- 不写"信号 vs 噪音"，写"哪些数据该看 / 哪些可以扔"。
- 不写"基因里就有"，写"从第一版就这么做了"。

---

## 三、AI 伪口语化词（命中即删）

> 拆一拆 / 拆开来看 / 掰开 / 盘一盘 / 聊一聊 / 捋一捋 / 盘点一下 / 划重点 / 敲黑板

替代：直接进入分析或观点，不需要预告动作。

---

## 四、AI 高频分析动词（全文 ≤ 1 次）

> 拆解 / 梳理 / 剖析 / 解构 / 聚焦 / 洞察 / 深耕 / 赋能 / 助力 / 践行 / 驱动 / 构建 / 打造

偶尔 1 次可接受，全文 ≥ 2 次必须改为具体动作或省略。
- "我们来拆解一下" → 直接给拆解结果。
- "深耕这个领域 5 年" → "在这个领域做了 5 年"。

---

## 五、降维引导语（全文 ≤ 1 次）

> 说白了 / 本质上 / 归根结底 / 一句话概括 / 简单来说 / 换个角度看 / 从某种意义上说 / 不妨这样理解

每次降维之后必须有具体事实或机制支撑，不能用降维代替论证。

---

## 六、步骤承接 / 元叙事过渡句（清零）

> 把 X 做完 / 理完 / 弄清，再 / 就 / 然后 Y
> 先看 A / 接着看 B / 看完了 A 再说 B
> 上面说的是 X，下面说 Y

改写策略：
- 删掉前半句，直接从新事实或判断起笔
- 需要因果时用轻量黏合词（于是 / 而 / 不过 / 可 / 其实）替代

---

## 七、AI 高频关联句式（全文 ≤ 2 次）

> 一旦……就…… / 只有……才…… / 无论……都…… / 随着……的发展（深入 / 推进） / 正是因为……所以…… / 正是……才…… / 通过……来……

超出预算改为轻量条件句或直接铺事实。

---

## 八、戏剧化揭露修辞（清零，引文除外）

### 比喻框架
> 遮羞布 / 最后一块遮羞布 / 面具 / 画皮 / 伪装 / 外衣 / 皇帝的新衣 / 幌子 / 烟幕弹

### 动词框架
> 扯下 / 撕下 / 揭下 / 摘下 / 掀开 + 遮蔽物
> 剥开 / 戳穿 / 戳破 / 揭穿 / 撕开 / 揭开 + 表象 / 真面目 / 本质 / 真相 / 外壳 / 包装

替代：直接给事实和判断。

---

## 九、极值判断句式（全文 ≤ 1 次）

> 最……的地方在于 / 是
> 真正……的是
> ……之处在于
> 更……的是

即使保留 1 次也必须紧跟**可核对事实**，否则改为直接陈述。

## 十、伪学术腔高频词

### 场 / 场域化合物 → 替换
- 舆论场 / 话语场 / 权力场 / 博弈场 / 公共场域 / 意义场域 → "舆论"、"讨论"

### "叙事"分析标签（非文学批评语境）→ 替换
- 宏大叙事 / 国家叙事 / 官方叙事 / 主流叙事 / 元叙事 → "说法"、"口径"

### "摊开"系列 → 删
- 摊开来看 / 摊开来说 / 把 XXX 摊开来 → 删引导动作，直接给事实

### "登台"系列（非戏剧语境）→ 改写
- 初登台 / 正式登台 / 粉墨登场

### 二元套壳 → 改写
- 面子上是 X，里子是 Y

### 行话堆叠（全文 ≤ 1）
> 底层逻辑 / 顶层设计 / 认知升级 / 降维打击 / 赛道 / 红利 / 抓手 / 闭环 / 存量 / 增量 / 博弈 / 范式 / 路径依赖 / 权柄 / 话语权

## 十一、判断式 / 定义式陈述堆叠（全文 ≤ 3，每处紧跟机制 / 事实）

### 定性等价
> "A 是 B"（作为定性标签）→ 改为动作 / 事实描述

### 否定等价
> "A 不是 B" → 改为不对称对照或让步-推进

### 字典式定义
> 所谓 A 即 B / A 就是 B / A 本质上是 B → 改为功能 / 场景还原

### 绝对化副词
> 必然 / 显然 / 毋庸置疑 / 根本 / 绝对 / 无疑 / 一定 / 无可争议
> + 判断句 → 改为"多半 / 通常 / 至少在……情况下"

---

## 十二、路标词预算（全文 ≤ 2）

> 更关键 / 更要命 / 换句话说 / 事实上 / 值得注意 / 总之 / 与此同时 / 不难发现 / 综上所述 / 基于以上分析 / 总的来说

超出预算就删，不要靠路标词撑结构。

---

## 十三、过渡腔（每段开头自查）

> 首先 / 其次 / 再次 / 最后 / 总之 / 综上所述 / 与此同时

每段开头若以这类词起手，先尝试**直接给判断或事实**，删掉序号词。

---

## 十四、空范围词

> 大量 / 广泛 / 深入 / 全面 / 显著 / 有效 / 合理 / 科学 / 专业 / 重要 / 关键 / 核心

每次使用都问一句："具体到多少 / 多大范围 / 哪些场景"。答不上来就删或改具体数字。
