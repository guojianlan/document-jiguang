一. 付费课程

我打算制作一门课程，叫做《从零到一全栈开发 AI 应用》，采用腾讯会议分享的形式，在线演示一个 AI 产品从零到一的开发过程。

我会选择我最近做的一个项目 [aiwallpaper.shop](https://aiwallpaper.shop/) 进行演示。

以建房子的步骤进行类比：

1.   先打地基：装环境/装软件，项目初始化
2.   搭建砖瓦：写 api / 写 web 页面 / 创建数据库 / 读写数据 / 调用 AI 接口
3.   接通水电：web 页面与 api 联调，打通数据流
4.   室内装修：使用 UI 组件对页面进行美化
5.   宴请宾客：发布上线，绑定域名，让所有人都能访问

通过这个课程，你可以了解到一个 web 型产品是如何从零到一构建完成的。你会对应用开发这件事情有一个全局的认识。

但是我可能不会讲的面面俱到，一些技术名词 / 技术细节问题，需要你自行学习。

我会给你推荐合适的资料，遇到问题我也会尽量帮助你解决。

1.   《从零到一全栈开发 AI 应用》

计划分享时间：2024/01/10 晚上 7 点，腾讯会议在线分享的形式。

以 [aiwallpaper.shop](https://aiwallpaper.shop/) 这个项目为例，演示从零到一开发一个 AI 应用。主要内容包括：

*   搭建本地全栈开发环境。包括 fnm / nodejs / pnpm / pgsql 等工具的安装
*   使用 nextjs 框架初始化一个项目，组织项目目录
*   开发 api / web 页面，讲解服务端渲染 / 前后端分离两种模式的数据交互方案
*   使用 tailwindcss / tailspark / shadcn 等 UI 组件库美化页面
*   使用 supabase 在线数据库，通过 pg 库进行数据的读写
*   使用 clerk 实现 Google 登录
*   使用 vercel 部署项目，绑定域名，发布上线
