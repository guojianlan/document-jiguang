《这就是 MCP》是一本全面、系统、详尽的 MCP 指南，旨在为读者提供一线开发与使用参考。

本书涵盖 MCP 的方方面面，从基础概念到实战案例，再到生态系统构建。全书共 6 章，内容结构清晰。

*   第 1 章讲述了 MCP 的工作原理、诞生背景和应用场景；
*   第 2 章深入解析了 MCP 架构；
*   第 3 章和第 4 章分别通过实际案例讲解了 MCP 服务器和 MCP 客户端的开发；
*   第 5 章展示了如何在典型应用场景下使用 MCP 服务器；
*   第 6 章介绍了 MCP 的生态系统。

![Image 3](https://r2.trys.ai/imgs/eb5c93409020cbc14fda3c208817676a.png)

本书适合所有对 MCP 感兴趣的读者，包括软件工程师、产品经理、设计师和创业者等。无论你是想了解 MCP 的基础知识，还是想深入学习 MCP 的开发和使用，本书都能满足你的需求。

1.   内容全面

本书的出发点，是帮助读者全面、系统地了解 MCP，包括 MCP 的诞生背景、运作原理、协议架构、应用场景、生态系统。

1.   案例丰富

本书用了三章的篇幅，通过具体的案例来讲解 MCP 服务器开发、MCP 客户端开发和 MCP 服务器使用。融合了我过去一年半做过的几款 AI 产品。设计的案例包括：

*   flomo-mcp，开发一个记笔记的 MCP 服务器
*   heybeauty-mcp，开发一个试衣的 MCP 服务器
*   chatmcp，基于 MCP 实现 AI 对话助手的核心功能
*   thinkany，基于 MCP 实现 AI 搜索引擎的核心功能
*   podlm，组合 MCP 服务器实现 AI 播客生成器
*   copyweb，组合 MCP 服务器实现 AI 网页生成器

1.   大佬推荐

围绕 MCP 生态，我请到了圈内的几位大佬联袂推荐。我想对大佬们表示感谢，并聊一下我的邀请逻辑。（排名不分先后，围绕 MCP 生态展开）

*   MCP 的设计初衷是给大模型补充上下文，硅基流动是大模型平台的典型代表，因此我请到了硅基流动的创始人袁老师，看看在大佬眼中 MCP 是怎样的定位。

*   通过对 MCP 的考古研究，我发现 MCP 在 3 月份爆火，主要是靠智能体领域的明星产品 Manus 带起来的，因此我请到了 Manus 的创始人 Red 老师，看看在大佬眼中，MCP 解决了什么问题。

*   2025 年是智能体蓬勃发展的一年，Same 是 AI Coding Agent 领域的知名产品，也是最早通过 MCP 集成扩展的产品之一，因此我请到了 Same 的联创 John，聊聊他对 MCP 潜力的看法。

除了大模型平台和智能体产品以外，chatbot 类产品也在 MCP 生态内占重要位置。因此我请到了国产 MCP 客户端四小龙中的三位创始人，来聊聊他们对 MCP 的看法。

*   ChatWise 是一款国产大模型客户端，其功能强大、体验流畅，深受开发者喜爱。它支持联网检索、画图、Artifacts、MCP 等多种功能。作者低空老师是一位开发能力极强的独立开发者，更新节奏快、效率高。

*   Cherry Studio 是一款高颜值的大模型客户端，支持模型切换、绘图、提示词应用、MCP 等功能。创始人 Yinsen 老师是一位有技术敏感度的产品经理，两人的团队，用一年时间，把产品在 GitHub 做到了 3w 星，非常了不起。

*   DeepChat 是一款开源的大模型客户端，内置模型对话、联网检索、知识库管理等功能，并对 MCP 有良好的集成，支持以对话方式编排工作流。创始人谢老师是积梦智能 CEO、ThinkInAI 社区（前 GoCN 社区）发起人，在开源社区有极大的影响力。

MCP 的发展，离不开社区的推广与内容博主的布道，也需要企业内一线工程师的实践与反馈。因此我请到了这几个领域的几位代表来聊聊他们对 MCP 的看法与理解。

*   AJ 老师是国内最大的 AI 社区 “WaytoAGI” 的发起人，通过举办共学活动，推动 AI 技术在国内的落地与普及。MCP 爆火之后，MCP.so跟 WaytoAGI 社区联合举办了 MCP 创新应用大赛，取得了不错的反响。

*   黄益贺老师是 newtype 社群的主理人，是国内少有的几个“能讲清楚 AI 的 Why 和 How 的”科技博主，黄老师讲 MCP 的几期视频做得非常棒，逻辑清晰、内容详实，我看完大受裨益。
