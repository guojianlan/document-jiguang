## 一、它到底能做什么？

在 Cursor、Codex、Claude code等聊天框中输入 /，AI 就会自动识别出 Impeccable 提供的 21 个强大的设计指令。

比如像我下图一样调用Adapt的指令，adapt是用于适配不同设备/场景的设计体验。

![](https://mmbiz.qpic.cn/sz_mmbiz_png/4ZNIrm7H9UPaMU0986ichDtcHdsqnkCib58TSMa4oibxCkyM9QXTULybCY6ba8IkxKBIbpiamsbicHdUP5xkZLJ5pPibCfZuovoF0libOXUXX0bhN0/640?wx_fmt=png&from=appmsg#imgIndex=1)

21个指令如下，需要哪个就调用哪个。
- adapt: 适配不同设备/场景的设计体验
- animate: 加入有目的的动效与微交互
- arrange: 优化布局、间距与视觉节奏
- audit: 全面审计可用性、性能、可访问性等
- bolder: 让“过于安全”的设计更有冲击力
- clarify: 让文案、标签、提示更清晰好懂
- colorize: 增强配色表现力与层次
- critique: UX 视角的设计评审与反馈
- delight: 增加惊喜感与愉悦细节
- distill: 去繁就简，强化核心表达
- extract: 抽取可复用组件/设计令牌
- frontend-design: 产出高质量前端 UI 实现
- harden: 处理边界、容错、i18n 等稳健性
- normalize: 对齐设计系统与一致性
- onboard: 优化新手引导/空状态
- optimize: 提升加载、渲染、动画性能
- overdrive: 技术驱动的高表现 UI 效果
- polish: 发版前精修，处理细节与一致性
- quieter: 降噪，弱化过度强势的视觉
- teach-impeccable: 初始化你的设计上下文
- typeset: 排版与字体层级优化

## 三、其他设计skill推荐

### 1. UI Skills

链接：https://ui-skills.com

包含 15 个独立 skills：baseline-ui（Tailwind 设计一致性）、fixing-accessibility（无障碍审计）、fixing-metadata（SEO 元数据）、fixing-motion-performance（动画性能）、12-principles-of-animation（迪士尼动画原则）等。

### 2. Taste Skill

链接：https://github.com/Leonxlnx/taste-skill

包含 taste-skill（前端设计）、redesign-skill（升级现有项目）、output-skill（强制 AI 写完整代码）、soft-skill（让东西看起来和感觉昂贵。涵盖高级字体、宽阔的空白、层叠的卡片设计、流畅的弹簧动画以及漂浮导航。）。

### 3. Better Icons

链接：https://github.com/better-auth/better-icons

MCP server + skill，搜索并获取 20 万+ 图标，支持自动学习常用图标库，直接同步到项目 icons 文件。

### 4. UI Design Brain

链接：https://github.com/carmahhawwari/ui-design-
