# T-反AI味检查清单

文章发布或主题笔记成稿后，按本清单自检。`/critique` skill 自动跑的也是这套规则——本模板是给人工自查时用的简版。

适用场景：
- 文章草稿成型后、跑 `/publish-article` 之前
- 发布版定稿前、对外分发之前
- 复盘旧文判断是否值得重写

---

## 一、4 维度打分（每项 1-5 分）

| 维度 | 评分 | 证据（引一句最差的） |
|---|---|---|
| 信息密度 |   /5 | |
| 独特视角 |   /5 | |
| 可操作性 |   /5 | |
| 阅读节奏 |   /5 | |

**判定**：
- [ ] 任一维度 ≤ 2 → 必须返工对应段落
- [ ] 总分 < 14 → 整篇重写而非局部修补
- [ ] 全部 ≥ 4 → 通过

---

## 二、硬规则快速自检

数一下出现次数：

- [ ] "而是"句式（含变体：不是…而是 / 并非…而是 / 不在…而在 / 不只是…更是）：______ 次（**全文 ≤ 1**）
- [ ] 冒号 `：` ：______ 次（**全文 ≤ 2，仅限直接引语**）
- [ ] 路标词（更关键 / 更要命 / 换句话说 / 事实上 / 值得注意 / 总之 / 与此同时 / 不难发现 / 综上所述）：______ 次（**全文 ≤ 2**）
- [ ] 协作路标词（接下来我们将 / 我们先来看 / 让我们一起 / 不妨一起）：______ 次（**= 0**）
- [ ] 客服 / AI 身份口吻（作为 AI / 截至我的知识 / 希望这能帮助你）：______ 次（**= 0**）
- [ ] 二人称连续使用：______ 处（**= 0**，单次"试想 / 当你"全文 ≤ 1）
- [ ] AI 高频隐喻词比喻义（噪音 / 信号 / 底色 / 滤镜 / 解药 / 土壤 / 基因 / 路径 / 切片 / 透镜 / 窗口 / 缩影）：______ 次（**= 0**）
- [ ] 戏剧化揭露（遮羞布 / 面具 / 撕开真相 / 揭开本质）：______ 次（**= 0，引文除外**）
- [ ] 极值判断（最……的地方在于 / 真正……的是 / ……之处在于 / 更……的是）：______ 次（**全文 ≤ 1**）
- [ ] AI 伪口语化词（拆一拆 / 盘一盘 / 聊一聊 / 划重点 / 敲黑板）：______ 次（**= 0**）
- [ ] AI 高频分析动词（拆解 / 梳理 / 剖析 / 解构 / 聚焦 / 洞察 / 深耕 / 赋能 / 助力 / 践行 / 驱动 / 构建 / 打造）：______ 次（**全文 ≤ 1**）
- [ ] 降维引导语（说白了 / 本质上 / 归根结底 / 一句话概括 / 简单来说）：______ 次（**全文 ≤ 1**）
- [ ] 模板段（挑战 / 机遇 / 展望 / 综上所述）：______ 处（**= 0**）
- [ ] "一段一标题"（`##` / `###` 下只有 1 段）：______ 处（**= 0**）

**命中超过 3 项硬规则 → 阅读节奏自动 ≤ 2 分。**

---

## 三、结构性检查

- [ ] 开篇是否一句话就给出读者愿意读下去的判断？（不是"在……的时代"开场）
- [ ] 每段是否至少有一个具体事实 / 数字 / 案例 / 亲历，而不是只有抽象论断？
- [ ] 有没有一个明显是"作者本人才会这样想"的角度？还是换任何同类作者写都是这样？
- [ ] 读者读完能立刻做哪 1-3 件事？（写下来，写不出来 = 可操作性 ≤ 2）
- [ ] 句长是否长短交替？随机抽 5 段，数每段第 1、3、5 句字数，差距 ≥ 10 字才算合格
- [ ] 有没有"啰嗦但有人味"的细节？（具体时间、具体名字、具体场景、自嘲、犹豫）

---

## 四、返工动作

按上面命中的低分维度选对应动作：

| 低分维度 | 补救动作 |
|---|---|
| 信息密度低 | 找最虚的 3 句 → 补具体数字 / 案例 / 亲历，或直接删 |
| 独特视角弱 | 找 1 个"作者会怎么想得不一样"的反向角度，重写开篇判断 |
| 可操作性差 | 把每个抽象建议落到工具名 + 步骤 + 失败兜底 |
| 阅读节奏差 | 拆长句 / 删过渡词 / 塞 1-2 个生活化细节 / 加 1 个断点 |

---

## 五、二轮自审

返工后再问自己一遍：**"这段读起来还有哪些地方明显是 AI 写的？"**

简短列出剩余痕迹（节奏 / 模糊 / 套话），再改一轮，目标是"不再明显是 AI"。

二轮仍 ≤ 2 分 → 停手，标记为"需要人工重写"，不要无限自动改。
